Create `src/hooks/useRoulette.ts` implementing the spin animation physics per D-10, D-11, D-12.

Hook signature:
```typescript
interface UseRouletteReturn {
  canvasRef: React.RefObject<HTMLCanvasElement | null>;
  spin: () => void;
  isSpinning: boolean;
  winner: Restaurant | null;
  reset: () => void;
}

export function useRoulette(
  segments: Restaurant[],
  onComplete: (winner: Restaurant) => void
): UseRouletteReturn
```

Implementation details:

1. **Refs (NOT state) for animation values** -- per Architecture anti-pattern #2, never use useState for frame-by-frame values:
   - `angleRef = useRef(0)` -- current rotation angle in radians
   - `velocityRef = useRef(0)` -- current angular velocity
   - `animFrameRef = useRef<number>(0)` -- requestAnimationFrame ID for cleanup
   - `segmentsRef = useRef(segments)` -- keep segments in ref to avoid stale closure

2. **spin() function** (per D-10):
   - Sets initial velocity: `velocityRef.current = 0.3 + Math.random() * 0.2` (radians/frame, gives 5-8 second spin)
   - Sets `isSpinning` state to true (this IS a React state since it controls UI)
   - Sets `winner` state to null
   - Starts animation loop via `requestAnimationFrame(animate)`

3. **animate() function** (per D-11):
   - `angleRef.current += velocityRef.current`
   - `velocityRef.current *= 0.985` -- exponential deceleration friction
   - `angleRef.current %= (Math.PI * 2)` -- keep angle in 0..2PI range
   - Call `drawWheel()` to render current frame
   - If `velocityRef.current > 0.001`: schedule next frame with `requestAnimationFrame(animate)`
   - If `velocityRef.current <= 0.001`: animation complete
     - Calculate winner index from final angle
     - Set `isSpinning` to false
     - Highlight winner segment (per D-12): redraw wheel with winner segment in brighter color
     - After 500ms delay, call `onComplete(winner)` and set winner state

4. **Winner calculation:**
   - The pointer is at the TOP (12 o'clock position, per D-02)
   - Each segment spans `(2 * Math.PI) / segments.length` radians
   - Winner index = `Math.floor(((2 * Math.PI - (angleRef.current % (2 * Math.PI))) / (2 * Math.PI)) * segments.length) % segments.length`
   - This accounts for: wheel rotates clockwise, pointer is at top

5. **drawWheel() function:**
   - Takes: `canvas: HTMLCanvasElement, segments: Restaurant[], currentAngle: number, highlightIndex?: number`
   - **HiDPI handling (per RLT-03, Pitfall #4):**
     ```typescript
     const dpr = window.devicePixelRatio || 1;
     const rect = canvas.getBoundingClientRect();
     canvas.width = rect.width * dpr;
     canvas.height = rect.height * dpr;
     ctx.scale(dpr, dpr);
     ```
   - Center point: `cx = rect.width / 2`, `cy = rect.height / 2`
   - Radius: `Math.min(cx, cy) - 10` (10px padding)
   - **Pastel colors palette (per D-01):** Cycle through these for alternating segments:
     ```typescript
     const PASTEL_COLORS = [
       '#FFB3BA', // pastel pink
       '#BAFFC9', // pastel green
       '#BAE1FF', // pastel blue
       '#FFFFBA', // pastel yellow
       '#E8BAFF', // pastel purple
       '#FFD9BA', // pastel orange
       '#BAFFEF', // pastel teal
       '#FFC9DE', // pastel rose
     ];
     ```
   - For each segment:
     - Calculate start/end angles accounting for `currentAngle` offset
     - Fill with pastel color (or brighter version if highlightIndex matches)
     - Draw thin white border between segments (lineWidth 2, strokeStyle white)
     - Draw text: category emoji + restaurant name (per D-03)
     - Text rendering: save context, translate to segment midpoint, rotate to segment angle, fillText centered
     - Font: `bold 12px sans-serif`, fillStyle `#333` for readability on pastel
     - Truncate long names to fit segment (max ~10 chars + emoji)

6. **Pointer (per D-02):**
   - Draw AFTER wheel segments so it's on top
   - Triangle at top center pointing down
   - Fill with `#EF4444` (red-500), with a small white border
   - Size: ~20px wide, ~25px tall
   - Position: centered horizontally at top of wheel

7. **reset() function:** Sets winner to null, resets angle to 0, redraws wheel static

8. **useEffect cleanup:** Cancel any pending animationFrame on unmount via `cancelAnimationFrame(animFrameRef.current)`

9. **useEffect to redraw** when segments change (draw static wheel when new restaurants loaded)
